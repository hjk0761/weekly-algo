# Info
[쇠막대기](https://boj.kr/10799)

- 난이도: 실버 2
- 분류: 자료 구조, 스택

## 💡 풀이 방법 요약

스택 문제입니다.

`()`가 오는 경우는 레이저고, 그 외의 경우는 모두 막대기입니다.

레이저가 나올 때 늘어나는 갯수는 지금까지 쌓인 쇠막대기의 개수와 동일하다는 점을 이용하면 됩니다.

`(`가 나오는 경우 막대기를 쌓고, `)`가 오는 경우엔 레이저인 경우 이전까지 쌓인 막대기 수만큼 결과에 더해주고, 아니라면 쌓인 막대기 개수를 하나 빼면 됩니다.
