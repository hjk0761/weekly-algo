## 💡 풀이 방법 요약

간단한 dp 문제입니다.

n = 1 일때의 경우의 수는 1개(세로 한 타일),

n = 2 일때의 경우의 수는 2개(세로 2타일 혹은 가로 2타일)입니다.

이 값을 베이스로 하여, 점화식을 세울 수 있습니다.

가로 n의 바닥은 `가로 (n-1)인 바닥에 세로 한 타일을 더하는 경우` 혹은 `가로 (n-2)인 바닥에 가로 두 타일을 더하는 경우` 로 나눌 수 있습니다.

따라서 점화식은 `f(n) = f(n-1) + f(n-2)`가 되며, 이 값을 배열에 계산하여 n일때의 값을 반환하면 됩니다.
