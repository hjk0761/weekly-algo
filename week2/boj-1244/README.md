# Info
[스위치 켜고 끄기](https://www.acmicpc.net/problem/1244)

- 난이도: 실버 4
- 분류: 구현, 시뮬레이션

## 💡 풀이 방법 요약

간단한 시뮬레이션 문제였습니다.

n이 최대 100이므로 O(n^3)으로 풀어도 100만이라 2초안에 여유롭게 들어옵니다.

입력을 하나씩 받으면서, 남자면 해당 배수만큼 토글(O(n)), 여자면 해당 칸 중심으로 한 칸씩 넓혀가며 좌우가 같은지 확인하고 같으면 토글(O(n))을 해 최종 시간복잡도는 O(n^2)입니다.

## 👀 실패 이유

처음엔 한 줄에 20개씩 출력해야 하는 걸 못봤고,

다음엔 32 line의 `result = []`가 `result = [[]]`였습니다.

이 때문에 항상 첫 줄에 추가 개행이 이루어져 출력 형식이 틀렸다고 나왔습니다.
